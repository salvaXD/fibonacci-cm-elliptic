## 📖 About

This application checks a mathematical identity that connects two different ideas: Fibonacci character sums and complex multiplication elliptic curves. The identity it verifies is \( S_p = -a_p(E) \). It uses number theory and computational math to perform this task.

You do not need programming experience to use this software. It runs on Windows and guides you through each step.

## 🛠️ What It Does

The software performs numerical checks of a formula related to advanced math topics such as:

- Fibonacci numbers and sequences
- Elliptic curves with complex multiplication (CM)
- Arithmetic dynamics and finite field computations
- Prime number distributions
- Frobenius trace calculations

It takes input data, runs calculations, and shows you results that confirm the link between Fibonacci sums and properties of elliptic curves.
